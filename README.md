# SmartFirstAidKit
- POSTECH 제 5회 무한상상 챌린지 대회 수상작(2등)
- 상황별 의약품 공급 스마트 구급상자 제작
- 2019년 3분기

# 주요 기능
- 사용자가 어플리케이션에서 증상을 선택하면 그 증상에 맞는 약품이 든 칸의 뚜껑을 열어 증상에 맞는 약품들을 떨어트려줍니다.
